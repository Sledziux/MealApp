## Technologies
Project is created with:
* React : 16.8.3
* React Native: https://github.com/expo/react-native/archive/sdk-34.0.0.tar.gz
* Expo : ^34.0.0
* Redux : ^4.0.4
* React Navigation : ^3.11.1

## Setup
To run this project, install it locally using npm:
```
$ npm install
$ npm start
```
